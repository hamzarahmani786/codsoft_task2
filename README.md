# CODSOFT Task 2 - Student Grade Calculator

## Description
A Java console application that calculates total marks, average percentage, and assigns grades for students.

## Features
- Input marks (0–100) for multiple subjects with subject names
- Calculates total marks and average percentage
- Grade assignment: A+ / A / B+ / B / C / D / F
- Pass/Fail status per subject
- Supports multiple students in one session

## Grade Scale
| Grade | Range  |
|-------|--------|
| A+    | 90–100 |
| A     | 80–89  |
| B+    | 70–79  |
| B     | 60–69  |
| C     | 50–59  |
| D     | 40–49  |
| F     | Below 40 |

## How to Run
```bash
javac StudentGradeCalculator.java
java StudentGradeCalculator
```

## Technologies
- Java (Core)
- java.util.Scanner
- java.util.ArrayList
